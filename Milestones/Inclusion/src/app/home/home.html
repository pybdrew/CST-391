Welcome to Inclusion

Inclusion is a Christian-themed personal study web app that allows you to read, add, edit, and delete personal study notes tied to Bible verses and translations.

Browse Verses My Comments

Functionality Overview

  • View Bible verses across multiple translations
  • View, add, edit, and delete personal comments per verse
  • Filter comments by translation (e.g., KJV, ACV)
  • Sort and review comments efficiently

About Inclusion

Inclusion is built with an Angular front-end, providing an interactive UI for scripture study. The backend is implemented in TypeScript with Express/Node.js and uses a MySQL database. REST API endpoints handle CRUD operations for comments, ensuring seamless management of personal study notes.
